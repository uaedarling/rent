# Rent Manager AED

A simple, self-hosted PHP rent management system for UAE property managers.
Manage units, tenants, payments, and generate receipts. Includes email notifications, WhatsApp integration, and a tenant-facing payment ledger.

## Features

- 🏠 Unit management (add/edit/deactivate units)
- 👥 Tenant management with per-tenant ledger links
- 💰 Payment recording with printable receipts
- 📧 Automatic email receipts to tenants
- 🔔 Late payment detection & notifications
- 👤 Multi-user with admin/manager roles
- 🔐 Secure login with CSRF protection, brute-force lockout
- 🔑 Forgot password / password reset via email
- 📋 Admin user management
- ⚙️ Settings: SMTP, WhatsApp Cloud API, company details

## Requirements

- PHP 7.4 or higher
- MySQL 5.7+ / MariaDB 10.3+
- PDO and PDO_MySQL extensions
- A web server (Apache/LiteSpeed/Nginx) with PHP support

## Installation

### Step 1 — Upload files

Upload all files from this repository to your web hosting public directory (e.g. `public_html/rent/`).

Do NOT upload:
- `config.php` (it doesn't exist yet — will be created by the installer)
- `installed.lock` (created after install)

### Step 2 — Create a MySQL database

In your hosting control panel (cPanel etc.):
1. Create a new MySQL database
2. Create a MySQL user with a strong password
3. Grant ALL PRIVILEGES on that database to that user

### Step 3 — Run the installer

Open your browser and go to:
```
https://yourdomain.com/install.php
```

Fill in:
- Database host (usually `localhost`)
- Database name, user, and password
- Admin email and password (you'll use this to log in)
- Company name and timezone
- SMTP settings (optional, for email receipts)

Click **Install Now**. The installer will:
- Create all database tables
- Create your admin account
- Write `config.php`
- Create `installed.lock`

### Step 4 — Secure the installer

After successful installation, **delete `install.php`** from your server. It's no longer needed and leaving it accessible is a security risk.

### Step 5 — Log in

Go to:
```
https://yourdomain.com/index.php
```

Log in with the admin email and password you set during installation.

## Cron Jobs (Optional)

To enable automatic late payment notifications, add cron jobs on your server:

```bash
# Daily at 9am: send email reminders to late-paying tenants
0 9 * * * php /path/to/your/rent/cron/notify_late.php

# Daily at 9am: send WhatsApp reminders to tenants and summary to manager
0 9 * * * php /path/to/your/rent/cron/send_late_reminders.php
0 9 * * * php /path/to/your/rent/cron/manager_late_summary.php
```

These run daily and send reminders to tenants who are past their grace period.
Each script is safe to run multiple times — it will not send duplicate reminders on the same day.

## Forgot Password / Reset

If you forget your password:
1. Click "Forgot password?" on the login page
2. Enter your email — a reset link will be sent (requires SMTP to be configured)

If you're completely locked out (no SMTP set up):
1. Create an empty file called `admin_reset.enable` in the app root directory
2. Visit `reset_admin_password.php` in your browser
3. Enter your admin email and new password
4. **Delete `admin_reset.enable` immediately after use**

## Security Notes

- `config.php` is excluded from git via `.gitignore` — never commit it
- All forms are protected with CSRF tokens
- Passwords are hashed with PHP's `password_hash()` (bcrypt)
- Login is rate-limited: 5 failed attempts = 15-minute lockout
- Session cookies are HttpOnly and SameSite=Lax
- `install.php` and `reset_admin_password.php` should be deleted/disabled after use
